## [1.0.0] - 2024-08-21

### Release Summary

**Release 1.0**: Bugfish Image Downloader is the inaugural version of our intuitive image retrieval tool. Download web images effortlessly, explore subsites, select high-definition options, and enjoy a seamless experience. This release marks the end of active development, and no known bugs exist. While there are many excellent image downloaders available, Bugfish Image Downloader stands out for its web scraping functionality, offering a unique solution in a limited software category.

### Features

- Effortless web image downloading.
- Subsite exploration for comprehensive image searches.
- Option to select high-definition images with a width of at least 1000 pixels.
- A seamless and user-friendly experience.
- No known bugs; stable performance.
- Unique web scraping functionality within the image downloader category.
