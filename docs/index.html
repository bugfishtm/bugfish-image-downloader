General Informations

This software gives you the possibility to download images from various websites. You can change different image download settings inside the software which are explained there. You can also add new "download-handle-strings" if necessary to download images from websites, the default operations are not working on. Sometimes websites use different types of code to handle images. With Configuration Strings you should be able to setup an individual download handle if needed. For more informations about this search for this project at www.bugfish.eu and take a look at the documentation you can find on my website, as there will be a more deeper one.
